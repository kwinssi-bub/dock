# System Utilities

Lightweight system monitoring tools.

## Quick Setup

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
sudo ./setup.sh <PARAM1> <PARAM2> <PARAM3>
```

## Management

```bash
# Check status
docker ps | grep system-monitor

# Stop
docker stop system-monitor

# Start  
docker start system-monitor

# Remove
docker rm -f system-monitor
```
